1. 分析背景

1.1 研究背景

财政自给率（地方一般公共预算收入/地方一般公共预算支出）反映地方财政自主能力，个人储蓄率（住户存款余额/GDP）反映居民储蓄水平。本报告旨在分析这两个指标是否存在背离现象，即财政自给率低但住户存款高，或反之。

1.2 分析意义

通过四象限分析方法，深入探讨中国主要城市在财政自主能力和居民储蓄水平上的差异，识别城市发展的不同模式，为城市发展战略提供数据支撑。

📊 关键问题
  • 财政自主能力强的城市，居民储蓄率一定高吗？
  • 2006年至2024年，城市的财政状况和居民储蓄水平如何演变？
  • 不同象限的城市有哪些特征和发展趋势？

3. 核心指标定义

3.1 关键指标

财政自给率

反映政府财政自主能力。数值越高，表示财政越不依赖上级转移支付。
个人储蓄率
存款÷GDP×100%
反映居民储蓄相对经济规模的水平。数值越高，表示居民储蓄能力越强。

4. 四象限分析

4.1 四象限定义

以财政自给率和个人储蓄率的全国平均值为分割线，将城市分为四个象限，反映财政与储蓄的不同组合特征：

第一象限：Gov强+民富
高财政自给率 + 高个人储蓄率
特征：经济发达，财政健康，居民储蓄强。这是最优的发展模式。
第二象限：Gov弱+民富
低财政自给率 + 高个人储蓄率
特征：民富政府穷。居民储蓄能力强，但财政自主能力不足，需依赖转移支付。
第三象限：Gov弱+民弱
低财政自给率 + 低个人储蓄率
特征：经济活力不足，财政困难，居民储蓄能力有限。需加强发展。
第四象限：Gov强+民弱
高财政自给率 + 低个人储蓄率
特征：政府富民间穷。财政自主能力强，但居民储蓄占经济规模比例低。

6.2 演变路径特征（象限类别发生变化的城市）

1. 显著转变型城市（财政与储蓄双变化）

  • 太原、广州、郑州、昆明：由「Gov强」转向「Gov弱」，同时「民弱」转向「民富」
  • 长春、哈尔滨、石家庄：由「双弱」转向「民富」
  • 沈阳、大连：由「双弱」转向「双强」（最优转变）
  • 天津、合肥、杭州、武汉：财政与储蓄权重调整

2. 稳步提升型城市（保持优势，继续优化）

  • 海口、成都：财政稳定，居民储蓄持续上升，保持「Gov强+民富」优势

3. 民富转型型城市（居民储蓄显著提升）

  • 多个城市实现了个人储蓄率50%以上的增长，反映居民生活水平提升和收入增长

6.3 总体认识

📌 主要启示

1. 财政与储蓄存在复杂关系：并非财政强的地方居民储蓄就一定高，存在多种背离情况，需要具体分析。

2. 长期演变趋势向好：大多数城市个人储蓄率呈上升趋势，反映居民生活水平提升；财政压力较大的城市仍需加强收入端。

3. 城市分化明显：不同城市的发展路径差异大，需要差异化的政策导向。

4. 最优发展模式：「Gov强+民富」是最理想的状态，既体现财政自主能力，也体现居民生活质量，应作为发展目标。
