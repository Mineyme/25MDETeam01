💰 任务一：预算缺口计算与分析

1.1 任务描述

计算目标：
  • 计算每个城市每年度的预算缺口 gap = expend - income（亿元）
  • 计算缺口占GDP的比重 gap_to_gdp = gap / gdp

1.3 关键发现

📈 缺口分布特征：
从原始数据分析可见，多数城市 gap_to_gdp 集中于0.0~0.1，但部分西部城市（如拉萨）个别年份超过0.3，显著拉高整体分布。
📊 长期趋势：
剔除极端值后（10%~90%缩尾处理），中位数从2006年的0.022稳步升至2022年的0.045，表明地方财政缺口占GDP比重长期呈扩大趋势。
⚠️ 重要观察：
2020年缺口骤升至峰值（中位数0.060），可能与疫情冲击下支出刚性增长而收入下滑有关。2021年短暂回落，但2022年再次回升，显示财政压力仍存。
📉 预算缺口占GDP比重（gap_to_gdp）分布演变
分析文字说明：
• 从箱线图（含离群值）可见，多数城市 gap_to_gdp 集中于0.0~0.1
• 西部城市（如拉萨）个别年份超过0.3，显著拉高整体分布
• 缩尾处理后两者波动接近，表明正常情况下收支增速同步性较强
• 财政缺口呈现长期扩大趋势，特别是2020年疫情冲击期间

📈 任务二：收支增长率分析

2.1 任务描述

计算目标：
  • 对每个城市，计算财政收入 income 的年度增长率（百分比）
  • 对每个城市，计算财政支出 expend 的年度增长率（百分比）
  • 计算公式：增长率 = (当年值 - 上年值) / 上年值 × 100

2.3 关键发现

📊 波动幅度差异：
原始数据中，收入增长率标准差（15.9%）高于支出增长率（12.6%），主要由于个别城市收入剧烈波动（如资源型城市）。
🔄 收支同步性：
缩尾处理后两者波动接近（收入10.5%，支出9.2%），表明正常情况下收支增速同步性较强。
⚠️ 周期性冲击：
2009年和2020年收入增长率显著为负，支出增长率则保持正增长，体现财政的稳定器功能。
📉 财政收支增长率对比
分析文字说明：
• 原始数据中，收入波动幅度大于支出
• 缩尾处理后两者波动幅度接近
• 支出增长率的年度间差异略小于收入，反映支出刚性特征
• 2009年和2020年都出现收入急剧下降，体现经济周期冲击

🎓 综合结论

通过三个任务的深度分析，得出以下主要结论：

  1. 财政缺口普遍存在且逐年扩大： 2006—2022年间，多数城市财政支出持续超过收入，gap_to_gdp中位数从约0.022升至0.045，显示地方财政压力加剧。这一趋势反映了地方政府支出责任不断扩大而收入增长相对滞后的矛盾。
  2. 增长率波动受经济周期影响： 2009年和2020年收入增长率大幅下滑，支出增长率保持韧性，反映财政逆周期调节作用；支出波动略小于收入，体现支出刚性。这表明地方财政在经济下行期间仍需维持基本支出，形成自动稳定器效应。
  3. 极端城市分化显著： 拉萨等西部城市缺口比重持续高企且攀升，杭州等东部城市长期保持平衡甚至盈余，区域财政自给能力差异悬殊。这一差异既反映了经济发展水平差距，也体现了现行转移支付制度的作用。
  4. 经济规模与财政健康度相关： 从极端城市看，GDP较高的城市（如杭州）往往缺口较小或盈余，而GDP较低的城市（如拉萨）缺口较大。这可能与税基规模、产业结构及转移支付制度有关。未来可进一步探讨财政转移支付的精准性与地方财政可持续性。

基于以上分析，建议：

✓ 逐步调整地方政府支出结构，促进收支平衡
✓ 优化税收分享制度，提高地方税收自给能力
✓ 建立财政风险预警机制，防范系统性风险
✓ 推进地方政府债务管理的规范化
